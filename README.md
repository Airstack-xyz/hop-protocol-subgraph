## Development

Prepare

```bash
npm run prepare:polygon
```

Codegen

```bash
npm run codegen
```

Build

```bash
npm run build
```

Deploy

```bash
npm run deploy
```

Prepare, build and deploy

```bash
npm run build-deploy:mainnet
npm run build-deploy:polygon
npm run build-deploy:xdai
npm run build-deploy:optimism
npm run build-deploy:arbitrum
```
